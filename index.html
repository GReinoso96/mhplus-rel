Monster Hunter G Plus

Monster Hunter G Plus is an improvement patch for the Japanese version of Monster Hunter G on the PS2, it supports both the Japanese original and the latest English Patch by Amaillo.

Click to see the MHGPlus Web Manual
(Latest) Version 1.0

Changelog

  • Initial Release
  • Added a toggle between Analog and Portable controls.
  • Added a toggle between Randomized Village Quests and a Slotted system.
  • Added a setting to control camera speed.
  • Added a toggle for 4:3 or 16:9 FOV.
  • Apply this patch to a clean Japanese ISO or an English Remix v11 ISO.

Monster Hunter 1J Plus

Monster Hunter 1J Plus is an improvement patch for the original Japanese version of Monster Hunter on the PS2, it supports both the Japanese original and the V8 English Patch.

Click to see the MH1JPlus Web Manual
(Latest) Version 3.5

Changelog

  • Fixed an issue with the Slotted quest system where the 3rd quest in 4 Stars was overwritten by a random number.
  • Tutorial button display has been updated to R3.
  • Analog Mode has been updated with a camera system, hold L2 (Item Wheel button) to move the camera using the right analog stick.
  • Added a new setting to control the game's standard camera speed.
  • Fixed an issue where setting the camera to REVERSE1 or REVERSE2 would lock it vertically.
  • Fixed bowgun recoil.
  • Removed Widescreen scaling on the bullet counter display.
  • Apply this patch to a clean Japanese ISO or an English v8 ISO.
