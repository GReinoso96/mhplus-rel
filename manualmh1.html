Monster Hunter 1J Plus Web Manual

How to Apply

Use xdelta3 to apply the patch:

xdelta3.exe -d -s "Eng v11 MHG.iso" "MHGJPlus Eng v1.0.xdelta" "MHGJPlus.iso"

xdelta3.exe -d -s "Clean MHG.iso" "MHGJPlus Jpn v1.0.xdelta" "MHGJPlus.iso"

Alternatively you can try a GUI tool like DeltaPatcher

Anamorphic Widescreen

No (Default)

The game will display its elements in the original 4:3 Aspect Ratio.

Yes

Primary Quest HUD elements and FOV will be scaled for 16:9 Widescreen.

Control Schemes

Analog Mode (Default)

  • Right Stick: Attack
  • R3: Kick
  • Triangle: Chat
  • Select: Resize Map
  • D-Pad: Camera

Portable Mode

  • Triangle & Circle: Attack
  • Select: Kick
  • R3: Chat
  • L3: Resize Map
  • D-Pad & Right Stick: Camera
Victory Camera

Yes (Default)

The camera focuses on the slain target upon completing a hunt or slay quest.

No

The camera remains with the player upon completing a hunt or slay quest.

Camera Speed

Note this only affects the normal camera, it should have no effect on aiming or binocs.

Normal (Default)

The original camera speed.

Fast

Camera speed is multiplied by 1.5.

Faster

Camera speed is multiplied by 2.

Village Quests

Random (Default)

The quest list is randomized the same way as the original game.

Slotted

Randomizes quests in a per-slot basis, that means every single slot in a quest level is chosen from 1, 2 or 3 specific quests, this ensure that, for example, you always have access to a Velocidrome quest in 3 Stars.

Extra

Based on the Slotted system, adds new quests from the international release of MH1.
