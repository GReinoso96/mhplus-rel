Monster Hunter 1J Plus Web Manual

How to Apply

Use xdelta3 to apply the patch:

xdelta3.exe -d -s "Eng v8 MH1.iso" "MH1JPlus Eng v3.0.xdelta" "MH1JPlus.iso"

xdelta3.exe -d -s "Clean MH1.iso" "MH1JPlus Jpn v3.0.xdelta" "MH1JPlus.iso"

Alternatively you can try a GUI tool like DeltaPatcher or xdeltaUI:

Anamorphic Widescreen

When enabled, most Quest HUD elements will be scaled horizontally to avoid stretching in 16:9 displays.

The game's Field of View is also affected and the minimap display is shrunk.

This mode is meant to be used on emulators with increased rendering resolution, though all elements should remain readable in real hardware.

Control Schemes

Analog Mode is the original game's control scheme, attack using the right analog stick.

Portable Mode is based on the control scheme of Monster Hunter Freedom on the PSP, attack using Triangle and Circle.

Victory Camera

Setting the Victory Camera setting to No will skip the camera animation that happens when completing hunting quests.

Village Quests

The Random setting makes village quest randomization proceed normally, that is, every quest is chosen at random from a pool, with the only consideration being that it doesn't repeat.

The new Slots system instead randomizes quests in a per-slot basis, that means every single slot in a quest level is chosen from 1, 2 or 3 specific quests, this ensure that, for example, you always have access to 1 Velocidrome quest and 1 Yian Kut-Ku quest in 3 Stars.
