Monster Hunter G Plus Web Manual

How to Apply

Use xdelta3 to apply the patch:

xdelta3.exe -d -s "Eng v8 MH1.iso" "MH1JPlus Eng v3.0.xdelta" "MH1JPlus.iso"

xdelta3.exe -d -s "Clean MH1.iso" "MH1JPlus Jpn v3.0.xdelta" "MH1JPlus.iso"

Alternatively you can try a GUI tool like DeltaPatcher

Delta Patcher
Widescreen

No (Default)

The game's FOV will remain as designed for 4:3 Aspect Ratio.

16:9

The game's FOV will be adjusted for 16:9 displays.

Control Schemes

Analog Mode (Default)

  • Right Stick: Attack
  • R3: Kick
  • Triangle: Chat
  • Select: Resize Map
  • D-Pad: Camera
  • L2 + Analog: Camera

Portable Mode

  • Triangle & Circle: Attack
  • Select: Kick
  • R3: Chat
  • L3: Resize Map
  • D-Pad & Right Stick: Camera
Camera Speed

Note this only affects the normal camera, it should have no effect on aiming or binocs.

Normal (Default)

The original camera speed.

Fast

Camera speed is multiplied by 1.5.

Faster

Camera speed is multiplied by 2.

Village Quests

Random (Default)

The quest list is randomized the same way as the original game.

Slotted

Increases the amount of possible quests per rank to 8 and randomizes quests in a per-slot basis, that means every single slot in a quest level is chosen from 1, 2 or 3 specific quests, this ensure that, for example, you always have access to a Velocidrome quest in 3 Stars.
